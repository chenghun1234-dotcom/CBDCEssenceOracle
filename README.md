# CBDCEssenceOracle (GEPA API)

Global Essence Price AI (GEPA) is the world's first AI-powered global commodity price API with CBDC-native currency conversion.

## 🚀 Quick Start

```bash
npm install
npx wrangler dev
```

## 🛠 Features
- **CBDC-Native Pricing**: Direct KRW-C, USD-C, EUR-C conversion.
- **AI Predictions**: 7-day and 30-day market trend forecasting.
- **Settlement Engine**: Automated calculation of logistics, tariffs, and total costs.

## 🔗 API Endpoints
- `GET /v1/market/prices`
- `GET /v1/market/predict`
- `POST /v1/settlement/calculate`
